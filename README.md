## Тема: Розпізнавання рукописних цифр від 0 до 9
---
Проєкт реалізує систему розпізнавання рукописних цифр від 0 до 9 з використанням архітектури CNN. Навчання моделі відбувалося на датасеті MNIST вбудованого в Tensorflow.

##  Як запустити проєкт

1. Встановіть необхідні залежності:
```bash
pip install -r requirements.txt
```
2. Запустіть застосунок:
```bash
python app.py
```
Після запуску перейдіть у браузері за адресою, вказаною в терміналі (за замовчуванням http://127.0.0.1:7860).

## Структура проєкту
- app.py — веб-інтерфейс на базі Gradio.
- mnist_cnn.keras — попередньо навчена модель.
- train.py — скрипт підготовки даних та навчання моделі.
- requirements.txt — файл з необхідними для проєкту залежностями
